# 📖 StoryTime – Interactive Storytelling Platform

A full-stack web application that delivers an immersive interactive storytelling experience through branching narratives and secure user authentication. Users can register, log in, choose from multiple story genres, and enjoy personalized reading sessions with progress tracking.

Built with **Node.js**, **Express.js**, **SQLite**, and **JavaScript**, StoryTime demonstrates full-stack web development concepts including authentication, REST APIs, database integration, and responsive UI design.

---

## 🚀 Features

### 📚 Interactive Storytelling
- Five interactive branching stories
- Multiple story genres (Horror, Fantasy, Mystery, Sci-Fi, Thriller)
- Dynamic story progression
- Personalized reading experience

### 👤 User Authentication
- Secure user registration
- Login & Logout functionality
- Session management
- User profile support

### 📊 User Features

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/khan2234mehak/storytime.git

cd storytime
```

### Install Dependencies

```bash
cd backend

npm install
```

### Start Server

```bash
npm start
```

Open your browser:

```text
http://localhost:3001
```
